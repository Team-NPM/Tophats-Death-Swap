# Changelog

## Release Version 1

### 1.0.0

- Initial Version
- Added configuration menu
- Added configuration option to enable or disable PVP
- Added configuration option to allow ender pearl usage. Can be turned off, on, or removed after 4 seconds of life
- Added configuartion option to reset advancements on game start
- Added configuration option to give players longer time on the first swap to get setup. Use to prevent cheap fall/lava traps
- Added configuration option to change what the timer looks like. Can be off, on, or vague, which gives time in less or greater than times
- Added configuration option to set minimum and maximum swap times
- Added configuration option to set minimum distance between players on initial swap
- Added configuration option to set maximum distance from spawn players are teleported on initial swap
- Added swapping logic
- Added lobby setup on new worlds
- Added win counter

### 1.0.1

- Set vague timer to have colors based on how long until swap
- Added swap grace option to configuration menu (does not function yet)
